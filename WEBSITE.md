# Сайт SPN

Публичный сайт открывается по адресу, указанному в `PUBLIC_SITE_URL`. По умолчанию адрес автоматически берётся из домена MiniApp, например:

```text
https://wayspn.ru
```

Первый экран сайта — компактная регистрация. После создания аккаунта пользователь сразу попадает к выбору тарифа и оплате.

## Настройка

Добавьте или проверьте значения в `.env`:

```env
PUBLIC_SITE_URL=https://wayspn.ru
WEB_SESSION_DAYS=30
WEB_COOKIE_SECURE=true
```

Для рабочего сайта обязателен HTTPS. После обновления перезапустите приложение — таблицы `web_accounts` и `web_sessions` создадутся автоматически.

В личном кабинете доступно:

- регистрация по логину и паролю;
- автоматический вход после регистрации;
- покупка новой обычной подписки или подписки с антиглушилкой;
- продление конкретной подписки;
- покупка дополнительных пакетов ГБ;
- просмотр ключей, сроков и трафика;
- подключение ключа в Happ и короткая инструкция прямо в кабинете;
- история платежей;
- автоматическая проверка результата оплаты после возврата с ЮKassa.

Пароли хранятся только в виде `scrypt`-хэшей. Сессии хранятся в базе как хэши случайных токенов, а браузер получает защищённую `HttpOnly` cookie. Веб-пользователи отделены от Telegram-пользователей внутренними ID и не обязаны иметь Telegram.

## ЮKassa

В кабинете используется ЮKassa. В настройках ЮKassa должен оставаться активным webhook:

```text
https://wayspn.ru/webhook/yookassa
```

Уведомление о платеже дополнительно перепроверяется через API ЮKassa: проверяются фактический статус и сумма.

## Важно

Регистрация только по логину и паролю пока не позволяет восстановить забытый пароль. Следующим этапом стоит добавить email, его подтверждение и восстановление доступа.
