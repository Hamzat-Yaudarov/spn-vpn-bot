# Веб-панель Way SPN

Панель открывается по адресу:

```text
https://ваш-домен/admin
```

## Настройка

Добавьте в `.env`:

```env
BOT_USERNAME=WaySPN_robot
ADMIN_ID=ваш-telegram-id
ADMIN_PANEL_URL=https://ваш-домен/admin
```

После обновления перезапустите приложение. При запуске автоматически создастся таблица `discounts`.

Панель должна открываться только через HTTPS. В главном меню бота кнопка «Админ-панель» показывается только пользователю с `ADMIN_ID`. Сервер проверяет подписанные Telegram `initData`, поэтому переданный браузером ID нельзя подменить. Отдельный пароль панели не требуется.

## Возможности

- общая статистика пользователей, подписок и выручки;
- поиск пользователя по Telegram ID или username;
- просмотр подписок и последних платежей;
- добавление и снятие дней подписки;
- удаление подписки с отключением в Remnawave;
- создание и отключение промокодов;
- создание tracking-ссылок и просмотр их результатов;
- планирование процентных и фиксированных скидок.

Скидки можно назначать на всё, все подписки, обычные подписки, антиглушилку, конкретный тариф, все пакеты трафика или конкретный пакет ГБ. Если одновременно подходят несколько скидок, применяется самая выгодная для пользователя; скидки не суммируются.
